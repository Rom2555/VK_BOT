# VK Bot — Бот для знакомств в ВКонтакте

Бот для ВКонтакте разработан в рамках учебного проекта.
Помогает находить людей по возрасту, полу и городу, и сохранять их в избранное.

## Структура проекта

Проект состоит из следующих файлов и папок:

- **database/** — модули работы с базой данных  
  - `__init__.py` — инициализация пакета  
  - `manager.py` — менеджер БД (работа с пользователями, кандидатами, избранным)  
  - `models.py` — модели SQLAlchemy (таблицы: users, candidates, favorites)

- **.env.example** — шаблон файла окружения  
- **.flake8** — настройки линтера flake8  
- **.gitignore** — файлы, игнорируемые Git  
- **config.py** — конфигурация (токены, настройки подключения к БД)  
- **keyboard.py** — генерация клавиатур для VK API  
- **main.py** — точка входа, запуск бота  
- **README.md** — этот файл  
- **requirements.txt** — зависимости проекта  
- **user_bot.py** — логика обработки сообщений от пользователя  
- **vk_searcher.py** — поиск пользователей, городов и фото через VK API

## Как запустить

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создайте файл `.env` на основе шаблона:
   ```bash
   cp .env.example .env
   ```

3. Заполните `.env` своими значениями:
   ```env
   GROUP_TOKEN=ваш_токен_группы_вк
   USER_TOKEN=ваш_токен_пользователя_вк
   DB_HOST=localhost
   DB_PORT=5432
   DB_NAME=название_базы
   DB_USER=имя_пользователя
   DB_PASS=пароль
   ```

4. Запустите бота:
   ```bash
   python main.py
   ```

## Как работает

- Пользователь пишет `/start`
- Бот просит ввести возраст, выбрать пол и указать город
- Поиск идёт через VK API
- Каждого кандидата можно:
  - Пропустить (`Дальше`)
  - Добавить в избранное (`Добавить в избранное`)
  - Посмотреть список избранных (`Избранное`)
  - Начать новый поиск (`Новый поиск`)
